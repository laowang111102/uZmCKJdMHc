# 项目简介 python9

本项目是一个基于Python的Django和Vue3实现的商城项目源码，集成了完整的系统功能和详细的设计文档，旨在为用户提供一套结构清晰、功能全面的商城系统。

## 技术栈

- 后端：Python, Django, Mysql
- 前端：Vue3

## 系统角色

- 用户：浏览商品，进行购买等操作
- 管理员：管理商品，处理订单等

## 运行环境

- 架构：B/S
- 系统环境：Windows/Mac
- 开发环境：Pycharm, Python3.0, Django, Mysql, NodeJs

## 功能模块

- 商品展示
- 商品搜索
- 购物车
- 订单管理
- 用户管理
- 管理员控制台

## 部署步骤

1. 准备开发环境
2. 克隆源代码到本地
3. 配置数据库
4. 运行项目
5. 访问前端页面进行测试
